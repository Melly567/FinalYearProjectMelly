About the Project

This Online Shopper Predictor uses machine learning to predict whether a user will make a purchase based on their browsing behavior and session information. The prediction model is trained on a dataset of online shopping sessions with various features.

How It Works

The application uses a Random Forest classifier trained on the Online Shopper Intention dataset. The model analyzes patterns in user behavior to predict purchase likelihood.

Key Features Used for Prediction:

  • Administrative Pages: Number of administrative pages visited
  • Product Related Pages: Number of product-related pages visited
  • Session Duration: Time spent on different types of pages
  • Bounce Rate & Exit Rate: User engagement metrics
  • Special Day: Proximity to a special day (e.g., Valentine's day)
  • Visitor Type: New or returning visitor
  • Weekend: Whether the visit occurred during a weekend

Making Predictions

To make a prediction:

  1. Navigate to the "Make Prediction" page
  2. Fill in the browsing behavior details
  3. Click "Make Prediction" to see the result
